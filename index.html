Bu'rra — Infant Safety Detection

2025 · Graduation Project · Princess Nourah University
Overview

Bu'rra is an AI-powered mobile application built as a graduation project to help caregivers monitor infants for choking incidents in real time, combining computer vision with a mobile app and cloud backend.

Challenge

Infant choking incidents can happen silently and quickly, often when a caregiver isn't in the room. There was a need for an automated, non-intrusive way to detect early signs of distress and alert caregivers immediately.

My Role

I worked as part of a 4-person team, contributing across multiple areas of the project including data preparation, model evaluation, and application integration with Firebase and Flutter.

Solution

AI-powered infant safety detection app that analyzes facial and physiological cues to detect choking signs in real time. The system uses MTCNN for face detection, HSV for skin color change analysis, CNN for classification, and Firebase/Flutter for the application.

Tools Used
Python Flutter Firebase MTCNN CNN
Key Contributions
  • Collected and prepared training datasets
  • Supported model evaluation and testing
  • Integrated detection pipeline with the mobile app
  • Documented system testing and results
  • Presented technical findings to faculty supervisors
What I Delivered
  • A working real-time choking detection pipeline
  • A documented test suite covering face detection, color analysis, and classification
  • A final presentation summarizing methodology and results

2025 · Rawafed Tech Course
